#EchoUI
基于React的轻量型组件库

## 特性
EchoUI = UI组件 + 常用工具类

## 开始
### 项目结构
    -- 1. assets 静态资源
    -- 2. compoennts 组件库
    -- 3. docs 文档
    -- 4. page 页面
    -- 5. package.json 
    -- 6. README.md

## 开发

## 运行
    -- 1. 安装node
    -- 2. cnpm i 安装依赖
    -- 3. npm run server 项目运行
## 协议
EchoUI 完全开源免费，可用于商业项目。
但不得基于本库开发新的 UI 框架再对外发布或销售。
